========================================================
   유튜브 영상 요약기 (완전 무설치판) - 사용 설명서
========================================================

유튜브 주소만 넣으면 영상을 내려받아, 소리가 크고 활기찬
구간만 골라 약 10분짜리 요약본과 자막(SRT)을 자동으로 만듭니다.

이 폴더는 파이썬까지 통째로 들어있는 "무설치" 버전입니다.
따로 설치할 것이 없습니다. 압축을 푼 뒤 바로 실행하세요.
(CapCut 도 필요 없습니다.)


────────────────────────────────────────────────────────
[ 실행 방법 ]
────────────────────────────────────────────────────────

   실행.bat  파일을 더블클릭하세요.  →  끝!

  - "YouTube 영상 요약기" 창이 뜹니다.
  - 파이썬 설치도, 인터넷 다운로드(프로그램)도 필요 없습니다.
  - 처음 한 번은 창이 뜨는 데 몇 초 걸릴 수 있습니다.

  ※ 폴더 통째로 옮기거나 USB에 넣어 다른 PC(윈도우 64비트)에서도
     그대로 실행할 수 있습니다.
  ※ 영상 다운로드에는 인터넷 연결이 필요합니다.


────────────────────────────────────────────────────────
[ 사용법 ]
────────────────────────────────────────────────────────

  1. "YouTube URL" 칸에 영상 링크를 붙여넣습니다.
  2. "출력 폴더" 기본은 output 폴더입니다. (바꿔도 됩니다)
  3. 옵션
     - 목표 길이(분) : 요약본 길이. 기본 10분.
     - Whisper 모델 : 자막 정확도. small(권장) 추천.
                      클수록 정확하지만 느립니다.
     - 언어         : ko(한국어)
     - 화질         : 360/480/720/1080.
                      작을수록 다운로드 용량↓, 속도↑ (720 권장).
     - 피크 앞/뒤 확장(초) : 골라낸 구간 앞뒤로 더 붙일 시간.
     - 같은 장면 묶기 기준(초) : 고른 하이라이트끼리 원본 영상에서
                      시간차가 이 값보다 짧으면 "같은 내용"으로 보고
                      사이 구간까지 포함해 하나로 이어붙입니다.
                      (전환 효과 없이 자연스럽게 연결. 기본 8초)
     - 장면 전환 효과 + 효과음 : 시간차가 큰(서로 다른) 하이라이트가
                      이어질 때만 짧은 암전(화면전환)과 "휙" 효과음을
                      넣어 끊김을 부드럽게 합니다. (기본 켜짐, 해제 가능)
  4. "다운로드 & 요약" 을 누르면
     다운로드 → 분석 → 편집 순으로 진행됩니다.
  5. 완료되면 출력 폴더에 두 파일이 생깁니다.
       제목_summary.mp4  (요약 영상)
       제목_summary.srt  (자막 파일)


────────────────────────────────────────────────────────
[ 자막(SRT) 수정하기 ]
────────────────────────────────────────────────────────

  자막은 영상에 새겨져 있지 않고 별도 .srt 파일로 나옵니다.

  - 메모장으로 .srt 를 열어 틀린 글자를 직접 고칠 수 있습니다.
  - CapCut/프리미어 등에서 영상과 .srt 를 함께 불러오면 표시됩니다.
  - 정확도를 높이려면 모델을 small -> medium 으로 올리세요.
    (기본 포함 모델은 base, small 입니다. medium/large 는 처음
     사용할 때 자동으로 인터넷에서 내려받아 models 폴더에 저장됩니다.)


────────────────────────────────────────────────────────
[ 폴더 구성 ]
────────────────────────────────────────────────────────

  실행.bat        더블클릭해서 실행
  사용설명서.txt   이 파일
  요약기_gui.py    프로그램 화면
  summarizer.py   요약 기능 본체
  python\         내장 파이썬 (건드리지 마세요)
  ffmpeg\         영상 처리 도구 (건드리지 마세요)
  models\         음성인식 모델 base, small
  output\         (요약 결과 저장 폴더)


────────────────────────────────────────────────────────
[ 자주 묻는 질문 ]
────────────────────────────────────────────────────────

Q. 창이 안 떠요.
A. 윈도우 64비트인지 확인하세요. 안 되면 D:\ 처럼 단순한
   경로로 폴더를 옮겨 보세요.

Q. 유튜브 다운로드가 안 돼요.
A. 인터넷 연결을 확인하고 잠시 후 다시 시도하세요.

Q. 영상 중간이 멈춰요 / 소리만 나와요.
A. 이 버전은 다시 인코딩하도록 고쳐졌습니다. 계속되면 화질을
   낮춰 다시 시도해 보세요.
